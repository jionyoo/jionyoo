## 🧱 Project

### 📸 Picme (셀프 증명사진 서비스)
집에서 촬영 → AI 검수 → 보정 → 인화/파일 다운로드까지 제공하는 서비스

- AI 기반 증명사진 검수 시스템 설계 및 운영
- 보정 상품 구조 (베이직 / 스탠다드 / 프리미엄) 기획
- 파일 다운로드 / 실물 배송 상품 구조 설계
- 장바구니 → 검수 → 주문 → 보정 → 다운로드 전체 플로우 설계
- CS / 검수 / 보정 프로세스 자동화 및 효율화
- Firebase 기반 데이터 구조 및 운영 시스템 설계
- 글로벌 확장 대응 (결제, 가격, 로컬라이징 구조)

👉 핵심:
운영 + 기술 + 기획을 하나의 시스템으로 통합
